기본 선택자 종류

  • 전체 선택자

    • HTML 페이지 내부의 태그를 모두 선택

    와일드 문자(*)를 사용하여 모든 태그에 적용

  • 태그 선택자

    • HTML 페이지 내부의 특정 태그를 모두 선택

    태그 이름이 선택자로 사용되는 유형

    선택자와 같은 이름의 모든 태그에 CSS3 스타일 시트 적용

    선택자 여러 개에 속성 적용하기 : 쉼표 사용

  • 아이디 선택자

    • 특정 id 속성이 있는 태그를 선택. 웹 표준에서 id 속성은 웹 페이지 내부에서 중복되면 안 된다는 규정이 있으므로 아이디 선택자는 특정 태그 하나를 선택할 때 사용

  • 클래스 선택자

    • 특정 클래스가 있는 태그 선택 (클래스는 중복 가능)

클래스 선택자

점(.)으로 시작하는 이름의 셀렉터

HTML 태그의 class 속성으로만 지정 가능

class 속성이 같은 모든 태그에 적용

아이디 선택자

id 속성의 목적 : 각 태그를 유일하게 구분

동일한 id 속성이 존재하지 않도록 HTML 파일을 작성하는 것이 바람직

아이디 선택자는 여러 태그 중 특정 태그에만 CSS 스타일을 적용할 때 적합
