CSS3 선택자 기본

CSS3 스타일시트 작성 방법 3가지

  1. style 태그에 시타일 시트 작성
  2. style 속성에 스타일 시트 작성
  3. 스타일 시트를 별도 파일로 작성
    • link나 @import로 불러 사용

CSS3 규칙

  • 태그에 적용되는 모든 스타일은 합쳐지고(Cascading)
  • 동일한 스타일은 순위가 높은 스타일이 우선 적용된다(Overriding)
  • 파일 => style 태그 => style 속성
